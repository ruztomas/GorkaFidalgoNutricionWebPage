# GorkaFidalNutricionWebPage

Web estática base para la carta de presentación de Gorka Fidalgo (nutricionista).

## Estructura

- `index.html`: página principal completa (hero, sobre mí, servicios, proceso, testimonio y contacto).
- `assets/img/gorka-hero.svg`: imagen placeholder de portada.
- `assets/img/gorka-about.svg`: imagen placeholder de sección "Sobre mí".

## Estilo

- Tailwind CSS vía CDN (sin build step).
- Tipografías: `Fraunces` (titulares) y `Manrope` (texto).
- Diseño responsive para móvil y escritorio.

## Cómo verlo

Abre `index.html` directamente en el navegador.

## Qué personalizar primero

- Sustituir placeholders por fotos reales conservando los nombres de archivo, o cambiando los `src` en `index.html`.
- Actualizar email, Instagram y ciudad en el footer.
- Ajustar textos de servicios y presentación al tono real de Gorka.
